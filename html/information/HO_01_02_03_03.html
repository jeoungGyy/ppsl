연금법 기준

  • 연계노령연금( 법제11조 )
    • 노령연금 수급권이 발생한경우(가입기간 10년이상)
      • 국민연금법에 의해 계산된 노령연금액
      • 국민연금법 제61조 및 제63조, 제51조제2항-제2항, 제52조 등
      • 대상 : 국민연금 가입기간 10년 미만
      • 연금액 : 기본금액 X 가입기간/20(가입기간 중 1년 미만인 월수는 매 1개월을 12분의 1년으로 계산)
    • 노령연금 수급권이 발생하지 않은 경우(가입기간 1년 이상~10년 미만)
    • 국민연금 가입기간이 1년 미만인 경우( 법 제4조제2항 )
      • 대상 : 연계기간이 20년 이상인 연금가입자 중 국민연금 가입기간이 1년 미만인 국민
      • 산정금액 : 국민연금법에 따른 반환일시금 + 영 제7조 이자
    • 반환일시금 이자( 영 제7조 )
      • 연계노령연금 지급연령이 된 날이 속하는 달까지의 이자： 국민연금법 시행령 제 50조 에따라 산정
      • 연계노령연금 지급연령 이후의 이자율 :
        • 연계노령 지급연령이 된 날이 속하는 달의 다음달부터 연계노령연금의 지급사유가 있는 날이 속하는 달까지의 개월 수에 그 기간 동안의 해당 연도마다 1월 1일 현재 전국은행이 적용하는 1년 만기 정기예금금리의 평균금리적용
  • 연계퇴직연금( 법 제12조 )
    • 퇴직연금(퇴역연금)의 수급권이 발생한경우
      • 직역연금법에 따라 계산된 퇴직연금액이나 퇴역연금
    • 퇴직(퇴역)연금 수급권이 발생하지 않은 경우(재직기간 1년 이상~10년 미만,단 군인은 1년 이상 ~20년 미만)
      • 대상：직역연금 재직기간이 10년 미만,군인연금은 20년 미만 (공무원·사학·별정우체국법('16년도부터)만 해당되는 급여산정기간)
      • 연금액：평균 소득월액 X 지급률(1년당 2%)X 재직기간 (1년 미만인 월수는 매 1개월을 12분의 1년으로 계산)
        ※ 퇴직일시금 수급권자가 연계퇴직연금 수급권자가 된 경우 퇴직일시금을 지급하지 않음
    • 직역연금 재직기간이 1년 미만인 경우(법 제5조제2항)
      • 대상 : 연계기간이 20년 이상인 연금가입자 중 직역연금 재직기간이 1년 미만인 공무원,사립학교교직원,군인 및 별정우체국직원
      • 산정금액
        • 퇴직일시금을 지급받지 않고 연계신청을 한 경우 : 직역연금법에 따른 퇴직일시금 + 영제8조 이자
        • 퇴직일시금을 반납하고 연계신청을 한 경우：반납금에 이자를 가산한 금액(퇴직일시금+이자)+ 영 제8조 이자
    • 퇴직일시금 등에 대한 이자
      • 퇴직일시금을 지급받지 않고 연계신청을 한 경우 : 퇴직일시금+이자
      • 퇴직한 전날이 속하는 달의 다음달부터 연계급여의 지급사유가 있는 날이 속하는 달까지 개월 수에 해당 연도마다 1월 1일 현재 전국은행이 적용하는 1년 만기 정기예금금리의 평균금리 적용(복리 계산)
      • 퇴직일시금을 반납하고 연계신청을 한 경우 :(퇴직일시금+이자)+이자
      • 반납금 등을 낸 날(분할 납부의 경우에는 마지막으로 분할 납부한 날)이 속하는 달의 다음 달부터 연계급여의 지급사유가 있는 날이 속하는 달까지 개월 수에 해당 연도마다 1월 1일 현재 전국은행이 적용하는 1년 만기 정기예금금리의 평균금리 적용(복리로 계산)
  • 연계노령유족연금( 법 제14조 )
    • 연계노령유족연금의 지급사유 ( 법 제13조 )
      • 연계노령연금 수급권자의 사망
    • 연계노령유족연금 수급권자 ( 법 제13조 )
      • 수급권자：국민연금법상 유족
      • 유족연금 수급권 발생 시기 :연계노령연금 수급권자가 사망한 날이 속하는 다음 달부터
    • 연계노령유족연금액 ( 법 제14조 )
      • 가입기간이 10년 이상인 연계노령연금 수급자가 사망한 경우 : 국민연금법에 따른 유족연금액
        가입기간이 10년 이상인 연계노령연금 수급자가 사망한 경우 : 가입기간, 연계노령유족연금액으로 구성
        가입기간 연계노령유족연금액
        10년 이상 20년 미만 기본연금액의 50%+ 부양가족연금액
        20년 이상 기본연금액의 60%+ 부양가족연금액
      • 가입기간이 10년 미만인 연계노령연금 수급자가 사망한 경우 : 10년미만 가입자에 대한 유족연금액(부양가족연금 제외)에 국민연금 가입기간을 10으로 나눈 비율로 적용한 금액
        가입기간이 10년 미만인 연계노령연금 수급자가 사망한 경우 : 가입기간, 연계노령유족연금액으로 구성
        가입기간 연계노령유족연금액
        10년 미만 기본연금액의 40%X 가입기간/10
  • 연계퇴직유족연금( 법 제15조 )
    • 연계퇴직유족연금액
      • 연계퇴직유족연금의 지급사유( 법 제13조 ): 연계퇴직연금 수급권자의 사망
    • 연계퇴직유족 수급권자 ( 법 제13조 )
      • 연계퇴직유족연금 수급권자 : 각 직역연금법 상 유족 (각 직역연금법 제2조 또는 제3조)
      • 유족연금 수급권 발생 시기 : 연계퇴직연금 수급권자가 사망한 날이 속하는 다음 달부터
    • 연계퇴직유족연금액 ( 법 제15조 )
      • 연계퇴직유족연금액 :연계퇴직연금액 x 직역연금법에 따른 유족연금 비율
        연계퇴직유족연금액 : 구분, 공무원연금법, 사학연금법, 군인연금법, 별정우체국법으로 구성
        구분 공무원연금법 사학연금법 군인연금법 별정우체국법
        관련조항 제55조 제1항 제42조
        ( 공무원연금법 제 55조 제1항 )         		제 26조 제 2항 제 1호 제25조의 3제4항
        유족연금비율 60% 60% 60% 60%
      • 직역연금 재직기간이 1년이상 20년미만인 연계퇴직연금 수급권자가 사망한 경우에 그 유족에게 직역연금법상의 유족일시금,유족연금부가금,유족연금특별부가금을 지급하지 않음.
      • 2009.12.31일 공무원연금법 및 사립학교교직원 연금법 개정에 의하여 해당 직역기관 신규자가 사망한 경우의 유족연금비율은 60%임.
  • 수급권자의사망에의한 급여정산 ( 법 제17조 )
    • 연계노령연금 및 연계퇴직연금 수급권자 사망
      • 급여지급청구를 하였으나 지급받기 전 수급권자가 사망한 경우 그 금액
      • 연금을 수급 중 해당월분 지급일 전에 수급자가 사망한 경우 그 금액
      • 수급권자가 급여지급 청구를 하지 않고 사망한 경우 지급기간 동안의 금액
    • 연계노령연금,연계노령유족연금
      • 국민연금법상 미지급급여 청구권자에게 지급 ( 국민연금법 제55조 )
    • 연계퇴직연금,연계퇴직유족연금
      • 연계퇴직유족연금 수급권자에게 지급 → 해당 직역연금법상 유족

    • 연계급여 미발생 사유
      • 수급연령에 도달한 자가 최소 연계기간을 충족하지 못했을 때
      • 수급연령을 넘어 직역에서 퇴직한 자가 최소 연계기간을 충족하지 못했을 때
        *반납금 또는 추납보험료를 납부하여 연계노령연금 및 연계퇴직연금을 지급받으려는 경우는 제외
      • 최소 연계기간을 충족하지 못하고 수급연령 전에 사망한 때
      • 최소 연계기간을 충족했으나 수급연령 전에 사망한 때
      • 60세가 넘어 임의계속가입하여 국민연금 반납․추납 중 임의계속가입을 탈퇴하였으나 최소 연계기간을 충족하지 못했을 때
    • 최소 연계기간
      • 직역 기관에서 퇴직일이 ’16년 1월 2일 이후인 경우 → 10년
      • 직역 기관에서 퇴직일이 ’16년 1월 1일 이전인 경우 → 20년
      • 연계기간에 군인연금 복무기간이 포함된 경우 → 20년

  • 연계급여미발생시처리(법 제19조)
    • 국민연금법상 반환일시금 ( 영 제7조 )
      국민연금법상 반환일시금 영 제7조 : 구분, 이자계신기간, 이자율로 구성
      구분 이자계신기간 이자율
      영 제7조의1 보험료 납부일이 속한 다음 달부터 지급연령이 된 날이 속하는 달까지의 이자는 국민연금법 시행령 제50조에 따라 산정 3년 만기 정기예금*
      영 제7조의2 연계노령연금의 지급연령이 된 날이 속하는 달의 다음 달부터 연계급여의 미발생사유가 있는 날이 속하는 달까지의 개월 수 1년 만기 정기예금
      (단기,평균 금리)
    • 연계신청을 하였으나 연계급여 수급요건을 충족하지 못한 상태에서 사망하는 경우 그 유족 등에게 지급해야 할 급여를 지급
      *2015.4.16.전에 납부한 연금보험료(반납금 및 추납보험료 포함)에 대하여는 2015.4월부터 3년만기 정기예금 이자율을 적용(2015.3월까지는 가입이력에 따라 3년 또는 1년만기 정기예금이자율 적용)

        퇴직일시금을 지급받지 않고 연계를 신청한 경우 : 구분, 이자계신기간, 이자율로 구성

        (복리,평균 금리)
      • 반납금등을 납부하고 연계를 신청한 경우 (퇴직일시금 + 이자) + 영 제8조
        반납금등을 납부하고 연계를 신청한 경우 : 구분, 이자계신기간, 이자율로 구성
